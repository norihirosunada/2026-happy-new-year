# ２０２６年　干支プロジェクト

## 概要

3Dモデルから特徴点を抽出し、平滑化された曲線を生成・可視化
馬の像を一筆書きで表現

## 更新情報 (2026-01-03)
プロジェクトを **Vue.js (Vite)** ベースに刷新しました。これにより、UIの操作性と開発効率が大幅に向上しました。

## 特徴
- **複数の接続モード**: 記録順、等高線、クラスタリング、最近傍探索による点と点の接続。
- **サンプリング制御**: 処理する点数の削減とダウンサンプリング。
- **高度な可視化**: Bloomエフェクト、影（チューブ化）、反射する床の設定。
- **画像保存**: ガイド枠を基準に高画質なスクリーンショットを保存。

## 開発環境のセットアップ

### インストール
```bash
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
```

## プロジェクト構成
- `src/App.vue`: メインレイアウトと状態管理。
- `src/components/ThreeViewport.vue`: Three.jsによる3D描画ロジック。
- `src/components/ControlPanel.vue`: パラメータ操作用UIコンポーネント。
- `src/utils/algorithms.js`: 接続アルゴリズム（K-Means, Nearest Neighbor等）のロジック。
- `vite.config.js`: Viteの設定。
